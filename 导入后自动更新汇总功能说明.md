## 功能说明

### 导入后自动更新逻辑

1. **用户导入账单**
   - 用户选择 Excel 或 CSV 文件进行导入
   - 系统解析文件并显示预览
   - 用户确认导入

2. **数据处理流程**
   ```
   确认导入 → 保存交易记录 → 更新账户余额 → 调用 renderHome() → 更新汇总显示
   ```

3. **自动更新的汇总项目**
   - **本月收入**：根据导入的收入类账单自动累加
   - **本月支出**：根据导入的支出类账单自动累加
   - **本月结余**：自动计算（本月收入 - 本月支出）
   - **账户总余额**：根据导入的账单自动更新

### 时间点说明

- **导入后立即更新**：用户点击"确认导入"按钮后，系统会立即刷新首页汇总
- **按当前月份统计**：所有导入的账单会根据其日期计入对应月份的统计中
- **自动计算结余**：本月结余 = 当月收入 - 当月支出

## 使用方法

1. 打开 `auto-accounting-h5/自动记账 V6 增强版.html`
2. 点击首页右上角的"导入账单"按钮
3. 选择账户和账单文件（支持微信/支付宝 Excel 格式）
4. 确认导入数据
5. 导入成功后，首页汇总会自动更新

## 注意事项

1. **月份匹配**：导入的账单会计入其交易日期所在月份的统计
2. **模板匹配**：请确保导入的账单格式符合微信支付或支付宝的标准模板
3. **账户选择**：导入前需要选择对应的账户，系统会自动更新该账户余额

## 测试建议

1. 准备测试账单文件（包含当月和往月的交易记录）
2. 导入前记录当前首页汇总金额
3. 导入包含支出 33 元的账单
4. 验证首页"本月支出"是否正确更新（例如从 0 变为 33）
5. 验证"本月结余"是否重新计算

## 技术实现

- **renderHome() 函数**：负责更新首页所有统计数据
- **getMonthlyStats() 函数**：计算指定月份的收支统计
- **confirmImport() 函数**：导入确认后触发数据更新流程
