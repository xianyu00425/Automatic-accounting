3. **开始测试**
   - 点击 "▶️ 开始测试" 按钮

4. **测试流程**
   ```
   步骤 1: 打开应用并记录初始状态

   ↓

   步骤 5: 验证收入更新（收入=1000，支出=33，结余=967）
   ```

### 方法 2：手动测试

1. **打开应用**
   - 文件：`auto-accounting-h5/自动记账 V6 增强版.html`

2. **记录初始数据**
   - 本月收入：¥0.00
   - 本月支出：¥0.00
   - 本月结余：¥0.00

3. **导入支出账单**
   - 点击"导入账单"
   - 选择账户（如：微信支付）
   - 上传测试账单文件
   - 确认导入
   - **验证：** 本月支出变为 -¥33.00，本月结余变为 -¥33.00

   - 再次点击"导入账单"
   - 上传收入账单
   - 确认导入
   - **验证：** 本月收入变为 +¥1000.00，本月结余变为 ¥967.00

5. **刷新页面**
   - 按 F5 刷新
   - **验证：** 数据保持不变

## 🔍 故障排查

### 问题 1：页面无法打开
**解决：** 检查文件路径是否正确，确保在正确的目录

### 问题 2：测试不执行
**解决：** 检查浏览器控制台（F12），查看是否有 JavaScript 错误

### 问题 3：数据不更新
**解决：** 清空浏览器缓存（Ctrl+Shift+Delete），重新打开页面
