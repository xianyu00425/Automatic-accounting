# Spec: PWA 改造

**Status**: approved
**Created**: 2026-04-18

## 背景
将单文件 HTML 记账应用改造为 PWA，使手机用户能安装到主屏幕使用，同时支持离线访问和自动更新。数据存储保持 localStorage，不同用户各自独立。

## 需求
1. 手机浏览器能触发"添加到主屏幕"提示
2. 安装后可离线使用
3. HTML 文件更新后，用户下次打开自动加载新版本
4. 桌面端正常使用，布局不变
5. 不同设备/浏览器的数据各自独立
6. 不支持 PWA 的浏览器中静默降级，不影响正常使用

## 约束
- 必须通过 HTTPS 或 localhost 部署
- manifest.json、sw.js、HTML 在同一目录
- 图标使用 PNG 格式，192x192 和 512x512 两个尺寸
- 不改变现有 localStorage 数据结构和 key 名称

## 验收标准
- [ ] 在 Chrome 移动端能触发"添加到主屏幕"
- [ ] 安装后断网打开应用正常显示
- [ ] 修改 HTML 后用户再次打开获得新版本
- [ ] 桌面 Chrome 打开正常使用
- [ ] Service Worker 注册不报错
- [ ] 不影响现有数据
