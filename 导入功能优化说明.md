# 自动记账 - 导入功能优化说明

## 📋 优化概述

本次优化针对账单导入功能进行了全面升级，提升用户体验和数据解析准确性。

## ✨ 新增功能

### 1. 模板选择功能
- **微信支付模板**: 严格按照微信支付账单格式解析
  - 字段：交易时间、交易类型、交易对方、商品、收/支、金额 (元)、支付方式、当前状态、交易单号、商户单号、备注
  - 时间格式：`2026-03-30 15:28:38` 或 `2026-03-30 15:28`

- **支付宝模板**: 严格按照支付宝账单格式解析
  - 字段：交易时间、交易分类、交易对方、对方账号、商品说明、收/支、金额、收/付款方式、交易状态、交易订单号、商家订单号、备注
  - 时间格式：`2026/3/31 12:08` 或 `2026/3/31 12:08:30`

### 2. 导入预览功能
- **预览弹窗**: 显示解析结果的详细预览
- **统计信息**: 
  - 成功解析数量（绿色）
  - 解析失败数量（红色）
  - 跳过行数（橙色）
- **预览表格**: 展示前 100 条记录，包含：
  - 交易时间
  - 交易对方
  - 金额（收支区分颜色）
  - 类型（收入/支出）
  - 分类
  - 状态标识

### 3. 时间格式优化
- **微信支付专用解析函数** (`parseWechatDate`):
  - 支持 `YYYY-MM-DD HH:mm:ss`
  - 支持 `YYYY-MM-DD HH:mm`
  - 支持 `YYYY/MM/DD HH:mm:ss`
  - 支持 `YYYY-MM-DD`
  - 支持 `YYYY/MM/DD`
  - 支持 Excel 日期序列号

- **支付宝专用解析函数** (`parseAlipayDate`):
  - 支持 `YYYY/M/D HH:mm`（无前导零）
  - 支持 `YYYY/M/D HH:mm:ss`
  - 支持 `YYYY-MM-DD HH:mm`
  - 支持 `YYYY-MM-DD HH:mm:ss`
  - 支持 `YYYY/M/D`
  - 支持 `YYYY-MM-DD`
  - 支持 Excel 日期序列号

### 4. 确认导入机制
- 用户必须点击"确认导入"按钮才会保存数据
- 支持取消操作，避免误操作
- 导入后显示成功提示

## 🔧 技术实现

### 核心函数

1. **selectTemplateAndImport(type)**
   - 验证账户和模板选择
   - 根据类型调用文件或粘贴导入

2. **updateTemplateDesc()**
   - 动态显示模板字段说明

3. **parseExcelFilePreview(data, accountId, template)**
   - Excel 文件解析（预览模式）
   - 存储结果到 previewData

4. **parseCSVFilePreview(content, accountId, template)**
   - CSV 文件解析（预览模式）

5. **parseCSVRowPreview(values, headers, accountId, fileType)**
   - CSV 行数据解析

6. **showPreviewModal()**
   - 显示预览弹窗
   - 生成预览表格

7. **confirmImport()**
   - 确认导入数据
   - 保存交易记录

8. **parseWechatDate(dateValue)**
   - 微信支付专用日期解析

9. **parseAlipayDate(dateValue)**
   - 支付宝专用日期解析

### 数据结构

```javascript
previewData = {
    transactions: [],  // 解析成功的交易
    errors: [],        // 错误信息
    skipped: 0,        // 跳过行数
    accountId: '',     // 账户 ID
    fileType: ''       // 文件类型（wechat/alipay）
}
```

## 📝 使用流程

1. 打开自动记账应用
2. 点击"导入账单"按钮
3. **选择账户**（必选）
4. **选择模板类型**（必选）：
   - 微信支付
   - 支付宝
5. 选择导入方式：
   - 从文件导入（Excel/CSV）
   - 粘贴数据（CSV 文本）
6. **查看预览**：
   - 检查解析结果
   - 查看统计信息
   - 检查错误提示
7. **确认导入**：
   - 点击"确认导入"按钮
   - 或点击"取消"放弃导入
8. 导入成功提示

## ⚠️ 注意事项

1. **必须选择模板**: 不选择模板会提示并红色边框警告
2. **时间格式匹配**: 严格按照模板格式解析时间
3. **预览限制**: 最多显示前 100 条预览记录
4. **错误显示**: 最多显示前 10 条错误信息
5. **数据验证**: 日期和金额为必填字段
